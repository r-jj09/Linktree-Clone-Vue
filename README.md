# hajdureka.com — Personal Linktree Clone

A custom-built link-in-bio page designed as a clean, accessible alternative to Linktree. Built with Vue 3 and deployed on Vercel.

## Features

- Categorized links (work & personal) with staggered entrance animations
- Fully accessible — semantic HTML, ARIA labels, keyboard navigable
- SEO-optimized with Open Graph and Twitter Card meta tags
- Responsive design with Tailwind CSS utility classes
- Optimized Lighthouse scores across performance, accessibility, and SEO

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | Vue 3 (Composition API) |
| Build tool | Vite |
| Styling | Tailwind CSS |
| Deployment | Vercel |
